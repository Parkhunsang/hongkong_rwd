# Hong Kong RWD

항공 예약 서비스를 주제로 제작한 반응형 웹 포트폴리오입니다.  
사용자가 메인 페이지에서 여행 정보를 탐색하고, 추천 여행지를 확인한 뒤, 예약 상세 페이지와 여행지 소개 페이지까지 이동하는 흐름을 구현했습니다.

PC와 Mobile 환경을 각각 고려해 레이아웃을 구성했으며, 실제 여행/항공 서비스 사이트처럼 보이도록 정보 구조와 UI 흐름을 설계하는 데 중점을 두었습니다.

## Project Overview
이 프로젝트는 반응형 웹 구현 능력과 콘텐츠 중심의 UI 설계 역량을 보여주기 위해 제작한 웹 포트폴리오입니다.

항공 예약 서비스의 구조를 참고하여 다음과 같은 사용자 흐름을 구성했습니다.

- 메인 비주얼과 추천 콘텐츠를 통한 서비스 소개
- 항공 예약 관련 UI 배치
- 여행지 상세 정보 페이지 제공
- 모바일 화면에서의 메뉴 구조 및 콘텐츠 재배치

## Key Features
- 반응형 웹 레이아웃 구현
- PC / Mobile 화면 분기 처리
- 메인 비주얼 슬라이드 UI
- 추천 여행지 및 인기 콘텐츠 섹션 구성
- 항공 예약 상세 페이지 UI 구현
- 여행지 소개 상세 페이지 구현
- 공통 헤더 / 푸터 분리
- Swiper를 활용한 슬라이드 인터랙션
- JavaScript 기반 콘텐츠 동작 구현
